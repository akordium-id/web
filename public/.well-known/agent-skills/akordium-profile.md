# Akordium Lab — Company Profile

## Who we are

Akordium Lab is a software development studio based in Surabaya, Indonesia. We help Indonesian businesses modernize legacy infrastructure through migration and greenfield system builds, with a focus on performance, cost efficiency, and zero-downtime deployment.

## What we do

- **PHP → Go migration.** Move legacy CodeIgniter 3 / Laravel systems to Go (Fiber, pgx) using Clean Architecture (handler → usecase → repository). Typical outcome: lower server cost, higher throughput.
- **MySQL → PostgreSQL migration.** Including time-series workloads on TimescaleDB (hypertables + compression policies) for GPS/IoT scale.
- **MIS & internal systems.** Management Information Systems, dashboards, and operational tools that replace spreadsheets and manual coordination.
- **Company-profile websites** as the entry point to larger business systems.

## Notable work

- **MIS-APAR** — CodeIgniter 3 → Go/PostgreSQL migration, ~50% reduction in server cost.
- **Orin GPS** — GPS tracking platform for 10,000+ devices, backed by TimescaleDB.
- **Bank Mega** — Automation & migration, 4× throughput improvement.
